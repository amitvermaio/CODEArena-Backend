# **Overview**

CODEArena-Backend is a powerful and flexible backend framework tailored for online coding platforms, supporting contests, problems, user management, and social interactions.  
It emphasizes **security**, **scalability**, and **seamless integration** across components.

# **Getting Started**

## **Prerequisites**

This project requires:

- **Programming Language:** JavaScript  
- **Package Manager:** npm

---

## **Installation**

Build CODEArena-Backend from source and install dependencies:

### **1. Clone the Repository**
```bash
git clone https://github.com/amitvermaio/CODEArena-Backend
```

### **2. Navigate to the project directory**
```bash
cd CODEArena-Backend
```

### **3. Install the dependencies**
```bash
npm install
```

### **4. Set up environment variables**
```bash
Copy the .env.example file to .env
```

---

## **Usage**

Run the project:
```bash
npm start
```

### **Testing**

CODEArena-Backend uses the {test_framework} test framework.

### **Run tests**
```bash
npm test
```
